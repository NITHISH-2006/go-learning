# Go Learning

This repository contains my Go learning journey.

Every folder contains small programs written while learning a specific concept. The goal is to learn by doing — small, focused examples with short notes.

## Topics Covered

- Variables
- Data Types
- Functions
- Conditionals
- Loops
- Arrays & Slices
- Maps
- Structs
- Methods
- Interfaces
- Error Handling
- HTTP Server
- JSON
- Goroutines
- Channels

## Repository Structure

```
01-basics
02-structs-methods
03-maps-slices
04-http
05-concurrency
06-error-handling
notes
```

## Running Examples

Run an example folder's `main.go`.

```bash
go run 01-basics/main.go
```

```bash
go run 04-http/main.go
```

## Purpose

This repo documents what I learned while studying Go for backend development and placement preparation.
